# JobLink Backend

A backend system for **JobLink**, a swipe-based job matching platform inspired by Tinder’s matching mechanism.  
Built with **Node.js, Express, MongoDB, JWT, TypeScript, Jest** and a clean layered architecture.

---

## Features (In Progress)

- User authentication (JWT Access & Refresh tokens)
- User profiles: skills, experience, preferences
- Job listings & company profiles
- Swipe-based matching (like Tinder)
- MongoDB storage with Mongoose
- Unit tests with Jest
- API documentation with Swagger
- Clean architecture: Controllers → Services → DB Layer
